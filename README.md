# Features

## Authentication with Email & Password
## Uploading Videos with Caption
## Compressing Videos
## Generating Thumbnails Out of Video
## Displaying Videos with Caption
## Liking on Posts
## Commenting on Posts

## Following Users
## Displaying Followers, Following, Likes & Posts of User
## TikTok Like UI
